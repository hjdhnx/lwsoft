## mylwsof 包说明
| 包内文件英文名        | 文件中文意思    |  作用  |
| --------    | -----: | :----:  |
| init.go      | 乐玩com初始化模块  |   用于初始化ole库并创建乐玩对象   |
| bitcolor.go      | 找图找色模块  |   用于windows窗口内找图找色   |
| mouse.go        |   鼠标操作模块 |   用于模拟鼠标的操作如鼠标左键单击，移动  |
| window.go        |  窗口操作模块  |   用于窗口设置，窗口绑定等操作  |
| system.go        |  系统操作模块  |   封装的乐玩系列系统操作如UAC控制  |
